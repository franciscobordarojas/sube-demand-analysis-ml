# Public Transport Demand Analysis (SUBE)

Proyecto de análisis de datos aplicado a la demanda de transporte público, utilizando información de validaciones del sistema SUBE para explorar patrones temporales y modelar la demanda mediante técnicas de regresión.

El proyecto se presenta como una pieza de portfolio: el análisis completo se encuentra ejecutado y los resultados pueden visualizarse directamente en la notebook incluida.

---

## Alcance del proyecto

El análisis cubre el flujo completo de un proyecto de Data Science orientado a datos reales:

- Análisis exploratorio de datos (EDA)
- Feature engineering temporal (día de la semana, franja horaria, etc.)
- Modelos de regresión (OLS y Ridge)
- Evaluación e interpretación de resultados
- Análisis de limitaciones del modelo

---

## Datos

El dataset corresponde a registros agregados de validaciones del sistema SUBE.

Se trabaja con información temporal y conteos de demanda, enfocándose en patrones de uso del transporte público a lo largo del tiempo.

Por cuestiones de tamaño y licencia, los archivos de datos no se incluyen en el repositorio.

---

## Notebook y resultados

La notebook principal del proyecto se encuentra en:

notebooks/Análisis_de_transporte_publico_Portfolio.ipynb


Incluye:
- análisis exploratorio con visualizaciones,
- construcción de variables explicativas,
- resultados de los modelos de regresión,
- métricas de evaluación e interpretación,

El proyecto está pensado para ser leído y evaluado directamente a partir de la notebook, sin necesidad de ejecutar el código.
